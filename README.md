## How to run locally
Make sure you have [Flutter](https://flutter.dev/docs/get-started/install) installed.

Clone this repository and run in device/emulator
```sh
$ git clone https://github.com/yosspang/belanja_pedia.git
$ cd belanja_pedia
$ flutter pub get
$ flutter run
```
